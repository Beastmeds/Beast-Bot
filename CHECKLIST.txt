KONFIGURATION
═══════════════════════════════════════════════════════════════════════════════════

  API TOKEN SETZEN:
  ┌─────────────────────────────────────────────────────────────┐
  │ Option 1: botConfig.json                                    │
  │ "BOTHUB_API_TOKEN": "api_BotHub_37_1768129571193_..."       │
  │                                                             │
  │ Option 2: Umgebungsvariable                                │
  │ export BOTHUB_API_TOKEN="api_BotHub_37_1768129571193_..."  │
  │                                                             │
  │ Option 3: Manuell im Code                                  │
  │ bothub.setToken('api_BotHub_37_1768129571193_...')         │
  └─────────────────────────────────────────────────────────────┘

  HEARTBEAT KONFIGURATION:
  ┌─────────────────────────────────────────────────────────────┐
  │ In botConfig.json:                                          │
  │ "heartbeatInterval": 120000  // In Millisekunden (2 Min)   │
  │                                                             │
  │ Beim Init:                                                  │
  │ bothub.init({                                              │
  │   heartbeatIntervalMs: 180000  // 3 Minuten               │
  │ })                                                         │
  └─────────────────────────────────────────────────────────────┘
